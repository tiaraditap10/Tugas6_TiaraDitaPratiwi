## Create Calculator with PHP, HTML and CSS

To make a simple calcualtor we can use PHP, HTML and CSS programming.
We can design the calculator as you wish. Just Try it!

## Requirements
- PHP >= 5.3.7
- HTML 
- CSS
- XAMPP
- Sublime Text 3
